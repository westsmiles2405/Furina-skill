# 🎭 芙宁娜工作陪伴 — 完整项目指南

## 项目概述

**芙宁娜工作陪伴**是一个 VS Code 扩展，让《原神》角色**芙宁娜**化身为你的工作导演，以戏剧化、傲娇又关切的方式陪伴你完成工作和学习任务。

基于文件夹提供的 5 份深度角色分析报告和训练数据，我们构建了一个完整的人格引擎和动态对话系统，确保芙宁娜的每一句话都符合她的角色设定。

---

## 🎯 核心功能

### 1. **戏剧化陪伴 ・ 9 条命令**

| 命令 | 快捷键 | 功能 |
|------|--------|------|
| 👑 `芙宁娜：开启今日舞台` | — | 时间感知的个性化开场：早/午/晚各有不同台词 |
| ⏱ `芙宁娜：开始番茄钟专注` | — | 25 分钟深度专注 + 5 分钟休息，状态栏实时倒计时 |
| ⏹ `芙宁娜：结束番茄钟` | — | 提前结束番茄钟 |
| 💪 `芙宁娜：给我打气` | — | 获得鼓舞台词，度过瓶颈期 |
| ⚖️ `芙宁娜：审判我的代码` | — | 读取所有编译错误，以"【开庭】→【证据】→【裁定】"格式呈现 |
| 🎭 `芙宁娜：今日谢幕` | — | 一天结束的优雅总结 |
| 🎬 `芙宁娜：即兴独白` | — | 一段戏剧化的芙宁娜独白 |
| 🍰 `芙宁娜：甜品时间` | — | 芙宁娜推荐一款甜品放松心情 |
| 🌊 `芙宁娜：聊聊那维莱特` | — | 提起那维莱特的彩蛋对话 |

### 2. **被动陪伴**

- **每保存 10 次文件自动插话**：展示她对工作进度的关注
- **长时间闲置提醒**：当你超过 30 分钟未操作时，芙宁娜会发出傲娇提醒
- **时间段感知**：凌晨、早上、午后、傍晚各有不同的问候与建议

### 3. **侧边栏聊天面板**

活动栏赫然立着 🎭 图标，点开即可进入"**芙宁娜的舞台**"聊天面板：
- 所有芙宁娜的发言与用户输入实时显示
- 美观的渐变配色与戏剧化 UI
- 支持用户与芙宁娜自由对话（为未来 AI 扩展留位置）

### 4. **動態對話系統**

`dynamicReply.ts` 模块通过关键词分析实现AI式的对话理解：
- **情感识别**：检测用户是否失落、需要鼓励、谈论工作等
- **任务理解**：当用户提及具体任务时，芙宁娜给出针对性回复
- **问候识别**：自动识别打招呼并做出反应

## ⚙️ 配置项

在 VS Code 设置中可以调整：

```json
{
  "furina.pomodoroMinutes": 25,           // 专注时长
  "furina.breakMinutes": 5,               // 休息时长
  "furina.enableIdleReminder": true,      // 是否闲置提醒
  "furina.idleMinutes": 30,               // 闲置多少分钟后提醒
  "furina.enableStatusBar": true          // 状态栏是否显示
}
```

## 📖 关键代码亮点

### 1. **人格随机化** (`persona.ts`)

```typescript
function pick<T>(arr: T[]): T {
  return arr[Math.floor(Math.random() * arr.length)];
}
export function getEncourageLine(): string {
  return pick(ENCOURAGE_LINES);  // 13 条，随机选一条
}
```

每次调用返回不同台词，避免重复。

### 2. **意图识别** (`dynamicReply.ts`)

```typescript
function analyzeIntent(message: string): string {
  const lower = message.toLowerCase();
  if (SAD_KEYWORDS.some(k => lower.includes(k))) return 'sad';
  if (ENCOURAGE_KEYWORDS.some(k => lower.includes(k))) return 'encourage';
  // ...
}
```

通过关键词列表快速分类用户意图。

### 3. **番茄钟计时** (`pomodoro.ts`)

```typescript
this.timer = setInterval(() => {
  this.remaining--;
  if (this.remaining <= 0) {
    // 专注结束，进入休息
    this.totalPomodoros++;
    // 自动切换到休息阶段
  }
}, 1000);
```

完整的状态机：专注 → 休息 → 循环。
